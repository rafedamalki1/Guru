This sample demonstrates the use of Microsoft Excel as an ActiveX Automation Server.
It extracts data from a Progress database, populates an Excel spreadsheet and creates
a graph in Excel. 

This sample program leaves Excel open.  You should close it manually when the program 
completes.

You must have the Excel application from Office 95 or Office 97 installed on your system
to run this example.  You must also connect to a sports database before running this 
sample.
