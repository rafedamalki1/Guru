This sample demonstrates the use of the Data Dynamics, Ltd ActiveBar 
control (actbar.ocx) and the Microsoft WebBrowser control that comes with Internet
Explorer.

It shows a simple Web Browser.  Operation should be familiar to anyone
who has used a web browser.  The ActiveBar control provides a toolbar
and menubar that allow you to navigate the web browser.  In addition, 
the toolbar and menubar can be undocked, or docked to different sides
of the browser window.  

The toolbar and menubar were created using the ActiveBar designer.

You must have the specified controls installed on your machine in order to run this sample.
