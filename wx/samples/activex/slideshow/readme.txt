This sample demonstrates the use of Microsoft Powerpoint as an ActiveX Automation Server.  
It shows a couple of slides in edit mode and then runs a slide show.

You must have Powerpoint from Office 95 or Office 97 installed on your system to run 
this sample.  Run the .p appropriate to the version of Office installed.


Files for this sample:

ppoint95.p
ppoint97.p
joke.ppt
test.ppt
auto.bmp
